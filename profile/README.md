# MyESC 组织

欢迎来到 MyESC GitHub 组织！我们专注于开源电调（ESC）项目开发，为无人机、航模和机器人社区提供高质量的无刷直流电机控制解决方案。

## 🎯 组织使命

MyESC 致力于开发创新的开源电子调速器（ESC）技术，通过开源协作推动电调技术发展，为全球无人机、航模和机器人爱好者提供可靠、高效的电机控制解决方案。

## 🚀 核心项目

### 主要项目

- **[BLDC-Halless](https://github.com/MyESC/BLDC-Halless)** - 无刷直流电机控制项目 ⭐ 4
- **[BLDC-Filter](https://github.com/MyESC/BLDC-Filter)** - 无刷直流电机控制项目 ⭐ 2
- **[MC_LIB](https://github.com/MyESC/MC_LIB)** - 核心库项目 ⭐ 10

### 项目特点

- **BLDC-Halless**: 专注于无感器控制算法，提供高效的无刷电机驱动方案
- **BLDC-Filter**: 电机信号处理和滤波优化，提升控制精度
- **MC_LIB**: 核心电机控制库，为其他项目提供底层支持

## 🛠️ 技术栈

我们使用的主要技术包括：

- **编程语言**: C（主要开发语言）
- **控制算法**: 无感器FOC控制、BLDC控制
- **硬件平台**: STM32、ARM Cortex-M系列
- **开发工具**: Git、GCC/Clang、调试器
- **协议**: PWM、UART、I2C、SPI通信协议

## 🤝 如何参与

我们欢迎所有形式的贡献！

### 贡献方式

1. **报告问题**: 发现 bug 或有改进建议？请在具体项目中创建 issue
2. **提交代码**: Fork 项目，创建分支，提交 Pull Request
3. **改进文档**: 帮助我们完善项目文档
4. **分享反馈**: 使用我们的项目并分享使用体验

### 开发指南

1. Fork 你感兴趣的项目
2. 创建功能分支: `git checkout -b feature/amazing-feature`
3. 提交更改: `git commit -m 'Add some amazing feature'`
4. 推送分支: `git push origin feature/amazing-feature`
5. 创建 Pull Request
